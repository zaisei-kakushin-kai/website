# 政府負担税の調査

消費税やガソリン税をはじめ、政府も様々な税を負担しています。
税制や財政の議論の場で話題にあがることはありませんが、政府財政の議論のために地方自治体の負担する公課を明らかにすることは重要だと考えます。

## 予算書について

地方自治体は議会に予算書を提出します。

予算書の粒度は団体によってことなり、テレビ受信料やケーブルテレビといった摘要まで公開している団体もありますが、消費税やガソリン税の税額までは記載されていません。

情報開示請求で政府の保有する情報の公開をお願いすることはできますが、地方議会に調査の陳情や請願を依頼することをおすすめしております。

## 陳情・請願の出し方

陳情・請願の出し方は議会によって異なります。

地方議会のウェブサイトに公開されていますので、ご確認ください。
