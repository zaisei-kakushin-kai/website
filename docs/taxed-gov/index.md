# 政府負担税の調査

消費税やガソリン税をはじめ、政府も様々な税を負担しています。  
財源のために増税しても政府の負担が増えてしまっては本末転倒です。

また、国債の発行で資金を工面した場合、消費税がかかるだけ、利払い費の負担が増えてしまいます。

政府の負担する消費税は財政悪化の一因である問題意識から、当会は政府の負担する消費税の調査を呼び掛けています。

## 政府の負担する消費税の概要

令和４年度の子ども家庭庁の概算要求明細表をみると、`ランチ代 3000円 * 1.1`や`ミネラルウォーター 150円 * 1.1`と記載されています。  
消費税が8%だった平成３０年の概算要求明細表には、`ランチ代 3000円 * 1.08`や`ミネラルウォーター 150円 * 1.08`と記載されています。

この1.1や1.08とあるのは、消費税分です。

相手が政府であっても消費税負担分は免除されませんので、政府がサービスや物品を購入するときも消費税を負担します。

土木工事をはじめ、電気代や公用車へのガソリンの給油、備品の購入やコンサル会社への発注といった、政府調達について同様です。

また、補助金や助成金についても消費税を加味した額が補助・助成されますので、同様のことがいえます。

さて、消費税分は循環して全て政府のもとに返ってくるように思えますが、そうとは言えません。

### 1. 利払費

建設国債をはじめ、国債を発行して得た資金で工面する事業は多数ありますが、この場合は消費税分、利払費が増えます。
例えば、高速道路の建設に1100億円かかるとなると、このうちの100億円分は消費税ということになります。

公債を発行して1100億円の建設費を工面すると、当然、100億円にも利子はかかります。

10年で償還する100億円の公債の利子率が2%でしたら、毎年2億円は余計に利払費がかかることになりますし、国債の発行残高も100億円分増えてしまいます。

国債投資家が国内にいれば、法人税や個人所得税として一部は政府のもとに戻ってくる可能性がありますが、外国に所在する投資家となるとそうとはいきません。

### 2. 地方消費税

地方消費税を考えると、「支払った消費税が戻ってくる」とは到底いえない状況があります。

令和2年4月1日以降は、消費税10%のうち、
- 2.2%が地方消費税
- 7.8%が国の消費税
- 7.8%の国の消費税のうち、地方交付税に当てられる部分が1.52%
となりました。

地方消費税の納付先は事業者の所在地によって決まります。
地方消費税の場合、発注した自治体のもとに返ってくる消費税額は少ない傾向があります。

以下、例をあげると、

- 国が東京都のコンサル会社に11億円の調査業務を発注すると、消費税は1億円。
  - 2200万円が東京都の地方消費税となり、1520万円が地方交付税となり、6280万円が残る計算です。

- 新潟県が神奈川県のシステム開発会社に11億円のシステム開発業務を発注すると、
  - 2200万円が神奈川県の地方消費税となり、1520万円が地方交付税となり、6280万円が国の消費税となる計算です。
  - 結果として、新潟県には、消費税が帰ってこない計算になります。

地方自治体は出来るだけ地元の業者に発注しようとしますが、大手企業は東京に事務所を構えていることが多いほか、デジタル系の案件はどうしても、東京の事業者に発注せざるを得えません。

地方公務員のなかには自分たちが負担した地方消費税が東京都のバラマキ政策の財源となっていることに不満を感じている人は多いようです。

なお、地方消費税は独自財源となるので、地方交付税交付金の交付額が減ってしまう場合があります。

### 3. 国の消費税

国の消費税についても、地方消費税と地方交付税を差し引いた6.28%しか戻ってきません。

## 政府の負担する消費税の概算
### 地方自治体の負担する消費税の概算

地方自治体の負担する消費税額の目安として、以下の式を提案します。

- A: 政府の歳出総額
- B1: 公債費
- B2: 人件費
- D: A - ( B1 + B2 ): 政府の歳出のうち、消費税が課税されたり、消費税を加味した額となっている部分
- D * 0.09 ≒ 政府の負担する消費税

補助金や助成金には消費税がかかりませんが、消費税を加味した額を支給するため、政府の負担する消費税の一部とすべきであると考えます。

軽減税率は食料品や新聞といった一部の商品に限られるため、概算方法としては妥当だと考えます。

地方消費税（地方消費税交付金）や地方交付税と比較して、消費税負担額のほうが多ければ、消費税を廃止しても現状の事業を維持することが出来る可能性があります。

### 国の負担する消費税の概算

消費税の1.52%が地方交付税の財源に充てられていますが、地方交付税を除外する必要があります。

- A: 政府の歳出総額
- B1: 公債費
- B2: 人件費
- B3: 地方交付税
- D: A - ( B1 + B2 + B3 ): 政府の歳出のうち、消費税が課税されたり、消費税を加味した額となっている部分
- D * 0.09 ≒ 政府の負担する消費税
