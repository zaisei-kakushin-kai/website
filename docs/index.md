# ようこそ

当ウェブサイトは、市民団体や政治に関心のある学生、立候補者のみなさまが気軽に使うことのできる企画を紹介しています。

## ウェブ予算書

予算書には政府の行う事業の予算が詳らかに記載されていますが、300ページを超えるPDFファイルを読み解いていくことは容易ではありません。

ウェブ予算書は、AIを使うことで誰でも気軽に見て、共有できる予算書のウェブ版を作ろうという企画です。

## 政府負担税

消費税やガソリン税をはじめ、政府も様々な税を負担しています。  
税制や財政の議論の場で話題にあがることはありませんが、政府財政の議論のために政府の負担する公課を明らかにすることは重要だと考えております。

自治体への請願や陳情をとおして、「政府の負担する税の調査」するためのテンプレートを公開しております。
